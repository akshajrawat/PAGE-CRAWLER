InstantSearch iOS 8.0 adds support for Algolia Swift client 9.x

InstantSearch iOS 8.0 is now compatible with the latest Algolia Swift client 9.x. This update includes the required migration and dependency upgrades to align with breaking changes introduced in Swift client 9.x. User Authentication and Memory

Agent Studio now supports user authentication and persistent memory across sessions. This update introduces secure user identification and conversation tagging, enabling agents to maintain continuity over time. By linking interactions to user IDs, agents can retain context, recall preferences, and build on past conversations to deliver more personalized and consistent experiences. Learn more in the Conversations and Memory documentation.

Prompt Suggestions in Chat

Prompt suggestions are now available in the chat experience. When using the InstantSearch chat component, users can enable or disable contextual prompt suggestions directly within the Agent Studio interface. The prompt guiding these suggestions can be edited, and users can configure the max number of suggestions as well as the maximum word count. This provides greater control and flexibility when designing chat interactions, helping deliver more relevant and consistent conversational experiences. Click here to learn more.

2 months ago

DocSearch v4.5 Adds Side-Panel Ask AI and Agent Studio Support

DocSearch v4.5 introduces a new side-panel Ask AI experience available in standalone mode or triggered from search in hybrid mode, enabling richer assistance without leaving the page. This release also adds Agent Studio support, allowing DocSearch to use the same agent architecture as other Algolia products, and expands framework support with VitePress compatibility and upcoming Docusaurus adoption. Ask AI thread depth controls let customers set a maximum conversation depth, improving response accuracy and cost predictability for longer side-panel or full-page conversations. Filter Suggestions Component

A new component, Filter Suggestions, is now available to enhance search and discovery interfaces with the Agent Studio. These suggestions are grounded in available filters within the index, ensuring accuracy and reliability. By proactively surfacing the most relevant filters, this feature helps users narrow results faster, explore content more efficiently, and achieve higher-quality search outcomes. Productivity MCP
